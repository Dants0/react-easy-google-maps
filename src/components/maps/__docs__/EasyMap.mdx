# EasyMap Component

The `EasyMap` component is a simple Google Maps wrapper built with `@react-google-maps/api`. It allows you to easily display a map with a marker, using only latitude and longitude as props.

### Usage

First, ensure you have a valid Google Maps API key, which you can obtain from [Google Cloud Console](https://console.cloud.google.com/).

Here's an example of how to use the `EasyMap` component inside your app:

```tsx
import React from "react";
import EasyMap from "../Map";
import { MapProvider } from "../MapProvider";

const Example: React.FC = () => {
  const latitude = -23.55052;
  const longitude = -46.633308;
  const apiKey = "YOUR_GOOGLE_MAPS_API_KEY";

  return (
    <MapProvider api_key={apiKey}>
      <EasyMap lat={latitude} lon={longitude} />
    </MapProvider>
  );
};

export default Example;
```
